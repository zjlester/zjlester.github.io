《Linux Web服务器管理入门》训前任务

发表于 2014-11-12 | 分类于 杂项其它 ， 课件下载 ， 资源分享

各位参加浙江省2014年信息技术学科带头人培训的学员：

感谢厚爱，邀请我再和大家进行一次交流。这一次交流我选的主题是《Linux Web服务器管理入门》，在开始见面交流之前，我想和大家先通过网络来聊聊，希望我们大家可以度过一个美好的上午。

首先，我知道，大家对这个主题可能会有一些复杂的情绪。班中的学员可能有大多数并没有承担学校里信息系统管理与维护的工作，即使承担了，可能也不涉及到服务器运维方面的具体任务，即使涉及了，也很可能只是在Windows Server上。Linux对于大家更多的可能只是一个纸面上、概念上的存在，所以这个主题对你们对我都是有挑战的。但我也是有信心的，因为在我的设计里，只要你依据我的思路去操作去思考，是可以在半天的时间里建构起对Linux基本操作的理解的，也是可以完成任务的。技术类主题的教学每个人都有自己的策略，我的方法是先以一个入门曲线较低的，具有实践意义的任务体验，来帮助学生建立概念体系——更重要的是建立信心，然后再慢慢延伸开去。所以这个成功的经验会比较的关键。

其次，说明一下课程的主要的任务，在这个半天时间里，大家需要依据规划请大家利用虚拟机安装一个Linux发行版本（我们会在Ubuntu Server 14.10）为例，然后在这个发行版本上从无到有建构起Apache、PHP、MySQL服务器环境，再接着，我们会一起在这个服务器环境里安装一个WordPress博客系统（你看到的这个博客系统，就是WordPress）。这里说明一下，之所以不选择桌面版的Linux主要的考虑一是虚拟机环境下效率不高，二是减少大家的干扰，桌面系统可能会给大家更多新奇感，但如果仅以GUI的方式去使用Linux，反而会影响大家理解与学习的进程。

第三，为了让大家周六上午一来就可以进入状态，我强烈地要求大家先做一些准备工作。先在自己的电脑上安装好Oracle VirtualBox（建议4.3.18版本，这是当前发布的最新版本）。其次下载好Ubuntu Server 14.10的光盘镜像，另外，为便于使用，推荐大家安装一款Windows下的ssh客户端（我推荐使用Xshell，因为我到时会使用这一款，当然你愿意去其它的，比如putty，也完全可以）。为了方便大家准备，我将链接放在下面：

  • VirtualBox 4.38:https://www.virtualbox.org/wiki/Downloads（选择VirtualBox 4.3.18 for Windows hosts ）
  • Ubuntu Server 14.10:http://www.ubuntu.com/download/server/
  • Xshell: http://www.onlinedown.net/soft/36383.htm
  • WordPress :http://cn.wordpress.org
