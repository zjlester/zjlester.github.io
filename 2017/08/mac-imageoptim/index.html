Mac软件推荐：图像压缩ImageOptim

发表于 2017-08-09 | 分类于 教育软件

在之前，我曾推荐过几个可以执行图片压缩的在线工具，在Mac也有非常好用的单机版工具，就是ImageOptim。软件的使用非常简单，要做的就是打开软件，然后将需要压缩的图片拖动中窗口中（或者也可以单击窗口下方的“+”号图标），然后压缩就自然执行了，如图所示。 如果你需要尝试一下这个软件，可以到它官网搜索下载，地址是：https://imageoptim.com 。如果速度比较慢，也可以在工作室直接下载：http://yun.zjer.cn/index.php?r=studio/resources/info&sid=531&resId=5d0b95f20fba443696563307c59fbeda 。
