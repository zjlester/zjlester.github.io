Win005:演示增强工具ZoomIt简介及下载

发表于 2014-12-24 | 分类于 教育软件 ， 杂项其它 ， 资源分享

软件简介

ZoomIt是一个我经常使用的免费小软件，称它为演示增强工具，主要是它提供了便捷的屏幕放大、批注的功能，非常方便在讲解、演示时使用。软件最早由System Internals公司开发，后来大概口碑比较好，财大气粗可以任性的微软就将它收购了～

Zoomit非常小巧，500多KB的样子，复制100份也填不满半个U盘的样子，我个人在平时的教学、讲座以及网络课程录制时中是经常用的，受我推荐使用过的小伙伴反馈的评价是：“非常有用，非常简单”。

主要功能与使用

  • 屏幕放大并批注 运行软件后，按CTRL+1，进入放大模式，默认放大两倍，移动鼠标可以选择放大的区域，滑动鼠标滚轮，可以增减放大倍数。选定放大区域后，可以单击，此时画面静止，鼠标变成批注状态。按ESC即可退出。
  • 屏幕批注模式 按CTRL+2进入。与前一步的差异，直接静固当前屏幕画面，并进入批注状态。默认是红色的笔，如果要切换颜色，可以按快捷键R（红）、G（绿）、B（蓝）切换，按T进入文字输入状态，但只能输入英文。
  • 屏幕倒计时器 按CTRL+3,屏幕将出现一个白底红字的倒计时，默认为10分钟，可以在软件设置中的“Break”选项卡中设置。

《Linux Web服务器管理入门》训前任务

发表于 2014-11-12 | 分类于 杂项其它 ， 课件下载 ， 资源分享

各位参加浙江省2014年信息技术学科带头人培训的学员：

感谢厚爱，邀请我再和大家进行一次交流。这一次交流我选的主题是《Linux Web服务器管理入门》，在开始见面交流之前，我想和大家先通过网络来聊聊，希望我们大家可以度过一个美好的上午。

首先，我知道，大家对这个主题可能会有一些复杂的情绪。班中的学员可能有大多数并没有承担学校里信息系统管理与维护的工作，即使承担了，可能也不涉及到服务器运维方面的具体任务，即使涉及了，也很可能只是在Windows Server上。Linux对于大家更多的可能只是一个纸面上、概念上的存在，所以这个主题对你们对我都是有挑战的。但我也是有信心的，因为在我的设计里，只要你依据我的思路去操作去思考，是可以在半天的时间里建构起对Linux基本操作的理解的，也是可以完成任务的。技术类主题的教学每个人都有自己的策略，我的方法是先以一个入门曲线较低的，具有实践意义的任务体验，来帮助学生建立概念体系——更重要的是建立信心，然后再慢慢延伸开去。所以这个成功的经验会比较的关键。

其次，说明一下课程的主要的任务，在这个半天时间里，大家需要依据规划请大家利用虚拟机安装一个Linux发行版本（我们会在Ubuntu Server 14.10）为例，然后在这个发行版本上从无到有建构起Apache、PHP、MySQL服务器环境，再接着，我们会一起在这个服务器环境里安装一个WordPress博客系统（你看到的这个博客系统，就是WordPress）。这里说明一下，之所以不选择桌面版的Linux主要的考虑一是虚拟机环境下效率不高，二是减少大家的干扰，桌面系统可能会给大家更多新奇感，但如果仅以GUI的方式去使用Linux，反而会影响大家理解与学习的进程。

第三，为了让大家周六上午一来就可以进入状态，我强烈地要求大家先做一些准备工作。先在自己的电脑上安装好Oracle VirtualBox（建议4.3.18版本，这是当前发布的最新版本）。其次下载好Ubuntu Server 14.10的光盘镜像，另外，为便于使用，推荐大家安装一款Windows下的ssh客户端（我推荐使用Xshell，因为我到时会使用这一款，当然你愿意去其它的，比如putty，也完全可以）。为了方便大家准备，我将链接放在下面：

Flash中利用ActionScript调用Google Map

发表于 2014-11-02 | 分类于 程序与设计

看到一个Flash作品中使用Google Maps，也想自己试一下，发现Google Maps 其实也是有For Flash的组件的，所以真正做起来还是比较简单的，记录了一下，主要步骤如下：

下载Google Maps API for Flash

  1. 地址是：http://maps.googleapis.com/maps/flash/release/sdk.zip。下载解压后有两个SWC，名为“map_1_20.swc”用于Flash Professional，名为“map_flex_1_20.swc”的用于Flex。
  2. 载完毕之后将它们复制到Flash的Components目录，为便于管理，还是在下面建一个Google目录比较好。 在Win7中，Components目录位于：C:\Users\Administrator\AppData\Local\Adobe\Flash CS5\zh_CN\Configuration\Components，用户名与语言不同，路径中相应的目录名会有变化；

申请Google Maps API Key

  1. 填写Google Maps API的申请表单，地址在http://code.google.com/apis/maps/signup.html;
  2. 在申请KEY时需要填写一个你最终发布SWF文件的域名，这是比较关键的，申请后，记住你的Key代码（下面要用到）

创建使用环境

在Flash中新建一个文档，此时在组件框中会看到多出来一个名为Google的文件夹，将里面的Google Maps Library拖至舞台，舞台上会呈现一个蓝框。

编写代码

  1. 创建一个新层用来添加ActionScript代码，在需要添加代码的地方，插入关键帧，按F9进入脚本编写窗口；

  2. 示例参考代码如下： //调用类 import com.google.maps.LatLng; import com.google.maps.Map; import com.google.maps.MapEvent; import com.google.maps.MapType;

    var map:Map = new Map(); map.key = “申请到的KEY”; map.sensor = “true”; map.setSize(new Point(stage.stageWidth, stage.stageHeight)); map.addEventListener(MapEvent.MAP_READY, onMapReady);this.addChild(map); function onMapReady(event:Event):void { map.setCenter(new LatLng(29.137243,119.640899), 14, MapType.NORMAL_MAP_TYPE);}

最终显示的效果如图所示，这里地图中心的坐标（ 29.137243,119.640899）其实是ZNU正校门进去的那个环路的圆心，要获取一个地方的坐标有一个比较简便的方法，就是在Google Maps上选中那个点，选择右键菜单中的“Where is here”。
